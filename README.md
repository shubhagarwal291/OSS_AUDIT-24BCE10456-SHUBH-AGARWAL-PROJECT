## Project Overview

This project is a comprehensive audit of an open-source software system, focusing on both theoretical understanding and practical implementation. The selected software for this audit is **Git**, a widely used distributed version control system.

The objective of this project is to explore the philosophy of open source, understand how such software is structured and maintained in a Linux environment, and demonstrate practical skills through shell scripting.

The project is divided into two major parts:
1. Analytical study of Git (origin, licensing, ecosystem, and comparison)
2. Implementation of Linux shell scripts to interact with the system and automate tasks

---

## Objectives

- To understand the origin and purpose of open-source software
- To analyze the licensing model and ethical aspects of open source
- To explore how software is installed and managed in a Linux environment
- To develop practical skills in shell scripting
- To demonstrate automation using Linux commands
